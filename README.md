# Student Management System

This is a simple Java console-based application to manage student records.

## Features
- Add Student
- View Students
- Menu driven interface

## Technologies Used
- Java
- Object Oriented Programming (OOP)
- ArrayList

## Project Structure

Student-Management-System
│
├── Main.java
├── Student.java
├── StudentManager.java
└── README.md

## How to Run

1. Compile the program

javac Main.java Student.java StudentManager.java

2. Run the program

java Main
